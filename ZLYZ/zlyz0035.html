然后解决问题

1/安装git

2/新建本地空文件夹

3/在空文件夹内git bash

4/【git clone 复制https】

5/进入克隆出来的文件夹

6/初始化本地空文件夹

【设置用户名】git config --global user.name 'yyyyy'

【设置用户名邮箱】git config --global user.email 'yyyy@yyy.com'

git config --list 查看

7/打开隐藏文件【.git】--【config】

8/找到【url = https:// github.com/Yvaine54/Yvaine54.github.io.git】

改为【url = https://用户名:密码@github.com/Yvaine54/Yvaine54.github.io.git】

9/Done~一般就可以正常操作了


10/如果有二般，遇到push不上去，那就【git pull –rebase origin master】走一趟
