Git新建仓库并关联本地一条龙

    编号：0038
    日期：2020.05.08    分类：杂乱有章

1/在github上创建仓库

2/在本地文件夹里：【git init】 初始化，生成.git文件夹

3/【git remote add origin 粘贴复制的地址】，地址是github仓库地址，关联上远程仓库

4/修改本地隐藏文件夹.git文件夹里的gonfig，修改内容参考0035日志

5/【git pull --rebase origin master】将远程仓库文件down下来，例如Readme

6/【git pull origin master --allow-unrelated-histories】

7/【git branch --set-upstream-to=origin/master master】

88888888/理论上这样就可以正常使用了，但如果不行

9/检查一下【git config --list】看有没有自己的用户名和邮箱

10/【git config --global user.name 'yyyyy'】设置用户名

11/【git config --global user.email 'yyyy@yyy.com'】设置邮箱
